# Terminology — Media OS

## Актуальные термины

### Document (Документ)

Главный объект работы. Имеет `title` и `document_type` (свободный text). Все материалы и действия принадлежат документу.

### Action (Действие)

Шаг работы внутри документа. Имеет `title`, опциональную привязку к Material. **Не называется «задача»** в UI.

Два булевых состояния:

- **Done (Готово)** — `done = true`. Действие выполнено, остаётся в списке, зачёркнуто.
- **Focus (В фокусе)** — `today = true`. Действие попадает на экран Today.

### Material (Материал)

Вспомогательный объект документа: источник, файл, ссылка, заметка. Поля: `title`, `material_type`, `file_url_or_path`, `notes`. Файлы не загружаются — только текстовый URL/путь.

### Workflow Template (Шаблон workflow)

Набор типовых действий для `document_type`. Хранится в `workflow_templates_v2`. Поля: `document_type`, `action_titles[]`.

При генерации действий создаются **независимые** Action-записи — связь с шаблоном не сохраняется.

### Today (Сегодня)

Рабочая поверхность внимания. Показывает actions с `today = true`, сгруппированные по документам. Не замена таск-трекера.

### Focus (В фокусе)

Состояние action: `today = true`. Пользователь вручную включает на странице документа. На Today — «убрать из фокуса» снимает флаг.

### Done (Готово)

Состояние action: `done = true`. Отмечается checkbox на странице документа или Today.

## Устаревшие термины (не использовать в UI)

### Task (Задача)

Legacy-сущность из v0.1. Таблица `tasks` сохранена в БД, но **не используется** в интерфейсе v0.2.

### Status (Статус)

Legacy-модель: `new`, `decision`, `stuck`, `done`, `let_go`. Заменена двумя булевыми флагами action: **Done** и **Focus**. Статусные блоки («Требует решения», «Застряло», «Можно отпустить») удалены из UI.

### Project (Проект)

Legacy-сущность. Не используется в v0.2.

## Технические термины (не для UI)

- **user_id** — идентификатор пользователя Supabase Auth, только для RLS.
- **document_type** / **material_type** — свободный text, не enum.
- **workflow_templates_v2** — имя таблицы; в UI — «Шаблоны».
